Simple Image Slide permits users to create Image Slide Show.

== Description ==

Create Simple Wp Image Slider and add Multiple images.

== Installation ==

1. Upload 'WP Easy Image Slider' to the wordpress admin
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open WP Image Slider and add a slide
4. Upload Multiple Images
